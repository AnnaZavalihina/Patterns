Часть 1: Реализация структуры связного списка
Создайте класс для узла связного списка (LinkedListNode). Узел должен содержать следующие поля:
Значение узла (например, тип int).
Ссылку на следующий узел.

Создайте класс для связного списка (MyLinkedList). Список должен содержать следующие методы:
Метод для добавления нового элемента в конец списка.
Метод для удаления элемента из списка.
Метод для поиска элемента в списке.
Метод для вывода элементов списка.
Метод сортировки списка.

Часть 2: Тестирование
Создайте набор тестов для проверки корректности реализации связного списка. Ваши тесты должны включать в себя:
Добавление и удаление элементов.
Поиск элементов.
Проверку корректности сортировки списка.

Проведите тестирование и убедитесь, что ваша реализация работает корректно.
