Часть 1: Реализация хеш-таблицы
Создайте класс для элемента хеш-таблицы (HashTableEntry). Элемент должен содержать следующие поля:
Ключ (например, строка или целое число).
Значение (например, любой объект).

Создайте класс для хеш-таблицы (MyHashTable). Хеш-таблица должна содержать следующие методы:
Метод для добавления нового элемента по ключу.
Метод для получения значения по ключу.
Метод для удаления элемента по ключу.
Метод для проверки, содержится ли ключ в хеш-таблице.
Метод для проверки, является ли хеш-таблица пустой.
Метод для очистки хеш-таблицы (удаление всех элементов).

Часть 2: Тестирование
Напишите набор тестов для проверки корректности работы вашей реализации хеш-таблицы. Тесты должны включать в себя:
Добавление и удаление элементов в хеш-таблицу.
Проверку наличия или отсутствия элемента в хеш-таблице.
Проверку состояния хеш-таблицы и её размера.

Проведите тестирование и убедитесь, что ваша реализация работает корректно.
