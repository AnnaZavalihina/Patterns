Часть 1: Реализация стека
Создайте класс для стека (MyStack). Стек должен содержать следующие элементы:
Массив или связный список для хранения элементов стека.
Метод для добавления нового элемента в стек (Push).
Метод для удаления элемента из стека (Pop).
Метод для получения верхнего элемента стека без его удаления (Peek).
Метод для проверки, является ли стек пустым.
Метод для очистки стека (удаление всех элементов).
Метод для определения размера стека.
Метод для поиска определенного элемента в стеке и возвращения его позиции (если элемент не найден, вернуть -1).
Часть 2: Тестирование
Создайте набор тестов для проверки корректности реализации MyStack. Ваши тесты должны включать в себя:
Добавление и удаление элементов.
Получение верхнего элемента без удаления.
Проверку размера стека.
Проверку поиска элемента в стеке.

Проведите тестирование и убедитесь, что ваша реализация стека работает корректно и обеспечивает правильное выполнение всех операций.
