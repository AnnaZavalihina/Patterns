Часть 1: Реализация очереди
Создайте класс для очереди (MyQueue) на основе массива или связного списка. Очередь должна содержать следующие методы:
Метод для добавления элемента в конец очереди.
Метод для удаления элемента из начала очереди.
Метод для получения элемента из начала очереди без его удаления.
Метод для проверки, является ли очередь пустой.
Метод для проверки размера очереди.
Метод для очистки очереди (удаление всех элементов).
Часть 2: Тестирование
Напишите набор тестов для проверки корректности работы вашей реализации очереди и системы управления задачами. Тесты должны включать в себя:
Добавление и удаление задач из очереди.
Получение верхнего элемента без удаления.
Проверку размера очереди.
Проверку поиска элемента в очереди.

Проведите тестирование и убедитесь, что ваша реализация работает корректно.
